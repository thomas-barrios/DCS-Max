## üöÄ **Quick Setup**

### **Step 1: Download WinUtil**
```powershell
# Run in elevated PowerShell
iex (iwr 'https://christitus.com/win' -UseBasicParsing).Content
```

### **Step 2: Load DCS Configuration**
1. **Open WinUtil** (will appear as GUI window)
2. **Go to Config tab** ‚Üí Import Config
3. **Select** `1.3.0-winutil-config.json` from templates folder
4. **Apply configuration** (will show all selected optimizations)

### **Step 3: Apply Optimizations**
1. **Review selections** in each tab (Tweaks, Features, Applications)
2. **Click "Run Tweaks"** to apply registry and system optimizations
3. **Click "Install/Uninstall Features"** to modify Windows features
4. **Restart computer** when prompted

- ‚ö†Ô∏è **Backup important data** in case of issues
- ‚ö†Ô∏è **Test on non-critical system first** if possible

## üîÑ **Reverting Changes**

### **System Restore Method**
1. **Access System Restore**: `rstrui.exe`
2. **Select restore point** created before WinUtil
3. **Confirm restoration** and restart system

### **Manual Reversal**
Some tweaks can be manually reversed:
```powershell
# Re-enable Windows Defender
Set-MpPreference -DisableRealtimeMonitoring $false

# Re-enable Windows Update
Set-Service -Name wuauserv -StartupType Automatic

# Re-enable Cortana
Get-AppxPackage -name "Microsoft.549981C3F5F10" | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register "$($_.InstallLocation)\AppXManifest.xml"}
```

### **WinUtil Restore Features**
WinUtil includes built-in options to reverse some changes:
- **Undo Tweaks** button in the interface
- **Reset to Windows Defaults** for specific categories
- **Restore removed features** (some limitations apply)
